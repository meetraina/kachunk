Everything to buy and register — in order.

Exact domains, handles, accounts, and store fees, sequenced behind the one gate that matters. Nothing on this page has been purchased or registered. Total to first store launch: ~$139–189 one-time + ~$27/yr.

Gate G-1 — clear the name before spending a dollar. The Justia screen found zero "kachunk" marks, but a screen is not clearance. Run the free USPTO search at tmsearch.uspto.gov for KACHUNK and phonetic variants (KACHUNCK, KACHONK, KA-CHUNK) in classes 9 (software) and 42 (SaaS), check state registers via a knockout search, and ideally have a trademark attorney do a 1-hour opinion (~$300–600, optional but recommended before store fees). HIGH RISK if skipped: every dollar and every handle below is wasted if the name has to change.
